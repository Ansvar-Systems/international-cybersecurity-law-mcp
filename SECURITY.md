# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in the International Cybersecurity Law MCP, please report it through [GitHub Security Advisories](https://github.com/Ansvar-Systems/international-cybersecurity-law-mcp/security/advisories/new).

Do NOT create a public issue for security vulnerabilities.

## Scope

This MCP server provides advisory cybersecurity law intelligence only. It does not:
- Connect to or scan any infrastructure
- Access cloud provider APIs
- Store customer data
- Process authentication credentials

| Version | Supported |
|---------|-----------|
